Weather Site Name

The name of this website will be "Texas' Weather Place."

Weather Site Purpose

The purpose of this website is to show the weather of Temple, Fort Hood, and Austin TX. The homepage will show the current weather, along with a sun or a cloud to show if it is sunny or cloudy. The cloud will be raining if it is raining in those cities. People will also be able to get updates of how the weather is going to be for 24 hours and then it will have content related to the latest weather news of these cities. This site will be a place where people can get information in a timely manner.

Weather Site Domain Name

The domain name that I have chosen is "weather-place.com" which is currently available at Hostgator for $12.95/year.
